Продавать доступ к Hysteria2 VPN (TimeWeb 64.188.57.249, s-ui) через Telegram-канал + бот + мини-апп.

## Шаги реализации

### 1. s-ui API — ядро автоматизации
- [ ] Изучить API s-ui: эндпоинты создания/удаления клиента, получения конфига
- [ ] Написать обёртку `createUser(userId, days)` → возвращает Hysteria2 URI
- [ ] Написать `deleteUser(userId)` для истёкших подписок
- [ ] Протестировать на тестовом клиенте

### 2. Telegram-бот
- [ ] Создать бота через @BotFather
- [ ] `/start` → приветствие + кнопка «Открыть магазин» (ссылка на мини-апп)
- [ ] Webhook: при успешной оплате → вызов s-ui API → отправка конфига клиенту
- [ ] Напоминание за 3 дня до истечения подписки

### 3. Варианты оплаты
- [ ] **Telegram Stars** — встроено в Telegram Payment API, без ИП, комиссия ~30%
- [ ] **ЮКасса / Тинькофф** — рублёвая карта, нужен ИП или самозанятый
- [ ] **Cryptomus** — USDT/крипта, без документов
- [ ] Реализовать все три, клиент выбирает при оплате

### 4. Мини-апп (витрина + оплата)
- [ ] Базируем на `viacheslav-tma.vercel.app` (уже развёрнут)
- [ ] Страница тарифов с описанием
- [ ] Кнопка оплаты → выбор метода (Stars / карта / крипта)
- [ ] После успеха: конфиг + QR-код для импорта в приложение
- [ ] Инструкция по установке приложения (Hiddify, NekoBox)

### 5. Telegram-канал
- [ ] Создать публичный канал
- [ ] Закреплённый пост: что такое VPN + кнопка «Купить»
