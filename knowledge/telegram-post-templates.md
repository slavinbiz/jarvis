# Шаблоны постов для @santex_ai

Читать когда пишешь пост в канал или готовишь продукт на продажу.

## Стиль канала (общие правила)

- От первого лица, без пафоса, конкретные цифры и факты
- Никаких «экспертных» заявлений без доказательств — только личный опыт
- Длина: 5–15 предложений, без лишней воды
- Без markdown-разметки (никаких **жирный**, только тире и переносы строк)
- Заголовок отдельно от текста, короткий, с контрастом «сантехник / ИИ-результат»

## Шаблон: пост о продаже продукта (файл/гайд/шаблон)

Структура:
1. Личный факт (возраст/стаж) — устанавливает контекст «обычный человек»
2. Когда начал разбираться в ИИ и зачем (не «стать программистом», а прикладная цель)
3. Что уже сделал (коротко, списком не нужно — 1-2 предложения)
4. Мостик: «если получилось у меня — получится у любого»
5. Что внутри продукта — списком через тире (конкретика: сколько промптов, что за чек-лист)
6. Результат — что получит покупатель и за какое время
7. Цена + сравнение (дешевле фрилансера/агентства)
8. Оплата — ссылка на перевод (ЮMoney/карта) + просьба прислать скрин с пометкой для чека
9. Страховка: «не получится — я на связи, помогу»

## Пример (продан 2026-07-05, файл с промптами для сайта)

```
Мне 57. 25 лет паял трубы.

Полгода назад начал разбираться в ИИ. Не чтобы стать программистом — чтобы зарабатывать, не выходя из дома.

Сделал бота для биржи, мини-апп и сайт. Ни строчки кода не написал сам — всё через нейросети.

Если получилось у меня — получится у любого.

Собрал рабочие промпты в один файл:
— 6 готовых запросов для ChatGPT (копируй и вставляй)
— чек-лист по шагам
— ссылки на бесплатный хостинг и картинки
— личная инструкция от меня

Результат: за 2 часа — живой сайт-визитка на бесплатном хостинге, с кнопками, формой и контактами.

Без кода. Без английского.

Цена: 990 ₽ — меньше часа работы фрилансера.

Оплата: https://yoomoney.ru/to/41001294600036?sum=990
После оплаты пришли скрин в личку с пометкой «оплата за сайт», скину файл.

Не получится с первого раза — я на связи, помогу.
```

## Оплата — что выяснили (2026-07-05)

- Т-Банк «Приём платежей» — только для ИП/ООО, самозанятым недоступен
- ЮMoney-кошелёк доступен самозанятым без ИП. Короткая ссылка на перевод с суммой:
  `https://yoomoney.ru/to/<номер_кошелька>?sum=<сумма>` — комментарий/пометку ссылкой не зашить, просить писать в переводе или присылать скрин
- Чек в «Мой налог» самозанятый пробивает вручную после каждой продажи, вне зависимости от способа оплаты (карта/кошелёк) — автоматизации нет, если не подключать Продамус
- Продамус — нужен только для полной автовыдачи файла без участия автора (регистрация магазина + встроенная интеграция чеков НПД). Для старта не обязателен
