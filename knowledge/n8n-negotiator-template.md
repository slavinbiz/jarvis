# n8n Переговорщик — сессионный roleplay-бот со сбросом памяти

**Источник:** получен от Вячеслава 2026-06-03, файл «Переговорщик.json»

**Когда читать:** когда нужен сессионный бот (игра, квест, тест) где `/start` начинает новую сессию и стирает историю.

## Архитектура

```
Telegram Trigger
  → IF message == "/start"
      [да] → Chat Memory Manager (удалить всё) → Агент + "Память очищена"
      [нет] → Агент напрямую
  → Telegram (отправить ответ)
```

## Ключевой паттерн: сброс памяти по команде

```
Chat Memory Manager (mode: delete, deleteMode: all)
  ← подключён к той же памяти (Память Кати), что и агент
```

При `/start` удаляет все сообщения из буфера. Агент при следующем запросе начинает с чистого листа. Подходит для: игр, викторин, тестов, любого сессионного контента.

## Параметры памяти
- `contextWindowLength: 20` — окно 20 сообщений (больше чем обычно, нужно для длинной игры)
- Ключ: `chat.id`

## Что интересного в промпте игры
- 5 сценариев на выбор + свой
- Система угроз 1-10 с эмодзи (🟢/🟡/🟠/🔴/🚨/💥)
- Анализ после игры: ошибки + оценка по 100-балльной шкале + рекомендации
- Кодовая фраза "Стоп, Завершить текущий диалог..." для принудительного завершения
