- **Продление:** ~16-е число каждого месяца
- **Проекты:** Крипто-бот Pamp-Damp + VPN (3x-ui, работает)
- **Нагрузка:** CPU ~3%, диск 31% (4.6/15 GB) — ресурсы есть
- **Управление:** timeweb.cloud, аккаунт `ug02085`
- **Управление ботом:** `screen -S bot` / `screen -r bot`
- **VPN-панель (3x-ui):** https://64.188.57.249:4096/NDat7WCdMq9RsAOdLz/ (самоподписанный сертификат — принять предупреждение браузера)
  - Логин/пароль сменены пользователем на свои, не хранятся здесь
  - Управление: `x-ui` (CLI на сервере: start/stop/restart/status/settings)
  - Если забудешь пароль — сброс через `x-ui setting -username <user> -password <pass>` на сервере
  - **VLESS+Reality (TCP/443):** инбаунд порт **443** (изменён с 21533 — TimeWeb блокировал старый порт на уровне облачного файрвола). Перестал работать в июне 2026 — заблокирован нацоператорами на уровне DPI/TSPU (падал и на домашнем WiFi, и на 4G одновременно). Сервис x-ui оставлен запущенным, но клиентам не использовать
  - **Hysteria2 (UDP/443) — основной рабочий протокол с 2026-06-15:** установлен официальным скриптом (get.hy2.sh), systemd-сервис `hysteria-server.service`, конфиг `/etc/hysteria/config.yaml`, самоподписанный сертификат (CN=www.bing.com, 100 лет), masquerade на https://www.bing.com
    - Ссылка для клиента (v2rayNG на Android, v2rayN на Windows — импорт из буфера обмена):
      `hysteria2://53dbf1a75aa7d83992b63567cdf866c2@64.188.57.249:443/?insecure=1&sni=www.bing.com#VPN-Germany-Hy2`
    - Проверено и работает на 4G (мобильный интернет) — обходит блокировку Reality
  - **Важно: облачный файрвол TimeWeb** (отдельный от ufw/iptables на сервере!) по умолчанию открыт только на порт 22 (SSH). Все остальные порты (включая VPN) нужно вручную разрешать в панели timeweb.cloud → Облачные серверы → сервер → Безопасность/Файрвол. Открыты: TCP/443 (Reality, неактивен), UDP/443 (Hysteria2, активен). OS-firewall (ufw) при этом может быть выключен — это не поможет, если облачный файрвол блокирует
  - Клиент (v2rayN, Reality, старое): локальный прокси 127.0.0.1:10808 (SOCKS5+HTTP). Для браузера включить системный прокси в Windows: Параметры → Сеть и интернет → Прокси → "Использовать прокси-сервер" → 127.0.0.1:10808
