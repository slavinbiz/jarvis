# n8n Telegram Bot — шаблон с мультиагентностью

**Источник:** получен от Вячеслава 2026-06-03, файл «Навыки для вашего бота.json»

**Когда читать:** когда нужно будет сделать Telegram-бота на n8n с AI.

## Архитектура

```
Telegram Trigger → LangChain Agent → Telegram Send
                        ↑
              LLM (Gemini 3 Flash) + Memory (10 сообщений) + Tool (sub-agent)
```

- **LLM**: `google/gemini-3-flash-preview` через OpenAI-совместимый API (провайдер: Polza)
- **Память**: Buffer Window Memory, ключ = `chat.id` (изоляция по пользователю)
- **Инструмент**: Генератор промптов для Nano Banana Pro (вложенный агент со своим LLM)

## Ключевые паттерны

1. **Главный агент + инструмент-агент** — главный решает когда звать специализированный
2. **Память по chat.id** — каждый пользователь видит только свою историю
3. **HTML-форматирование** в system prompt — `<b>`, `<i>`, `<code>`, `<pre>` для Telegram

## System prompt бота (основная суть)

Бот «Катя» — полезный помощник с чувством юмора. Форматирует ответы HTML-тегами для Telegram. ОБЯЗАН использовать инструмент «Генератор промптов Нано Банана» при любом запросе на создание промпта для генерации изображений.

## Инструмент: Генератор промптов Нано Банана

Специализированный sub-agent для создания промптов для Nano Banana Pro (Google, на базе Gemini 3 Pro).

Ключевые правила промптов:
- Писать связными предложениями, как описание художнику
- Обязательные блоки: субъект, действие, локация, композиция/ракурс, освещение, стиль
- НЕ использовать «теги качества» (masterpiece, best quality, 4k) — это для диффузионных моделей
- НЕ указывать соотношение сторон и разрешение в тексте
- Текст на изображении заключать в кавычки
- Язык промпта = язык запроса пользователя
