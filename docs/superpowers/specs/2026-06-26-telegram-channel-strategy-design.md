## Позиционирование

**Кто ведёт:** Сантехник по найму, освоивший вайбкодинг с нуля — без IT-образования, без опыта в программировании. Реальные проекты: лендинг, Telegram Mini App, крипто-бот, Lead Machine, VPN-настройка.

**Для кого:** Самозанятые, мастера, малый бизнес (электрики, строители, репетиторы, мастера по ремонту). Люди, которые хотят зарабатывать с помощью ИИ, но думают «это не для меня».

**Главная идея:** *«Если сантехник может — можешь и ты»*. Никакой теории ради теории — только то, что приносит деньги или экономит время.

**Уникальное преимущество:** Автор — человек из той же среды, что и аудитория. Это доверие, которое сложно купить.

## Контент

### Три рубрики (чередуются, 1 пост в день)

**1. «Делаю в прямом эфире»** — build-in-public
- Показываешь текущий проект: скрин, 3-5 предложений что происходит, сколько времени ушло
- Не нужно объяснять детали — процесс важнее теории

**2. «Как это сделать»** — практические гайды
- «Как я сделал сайт для мастера за вечер»
- «Как настроить бота для приёма заявок»
- Конкретный инструмент → конкретный результат → реферальная ссылка

**3. «Деньги и ИИ»** — про монетизацию
- Сколько стоит такой бот на рынке
- Где искать первых клиентов
- Какие инструменты окупаются
- Самый «горячий» контент для аудитории

### Три фазы развития

**Фаза 1 — 0 клиентов (сейчас)**
- Показываешь уже готовые проекты как портфолио
- Исследуешь рынок вслух: цены, спрос, конкуренты
- Честно без выдумок

**Фаза 2 — поиск первого клиента**
- Документируешь процесс поиска: разместил оффер → 0 ответов → вот что менял
- Живой контент, люди ценят его больше историй успеха

**Фаза 3 — первый заказ**
- Настоящий build-in-public: взял заказ, показываешь процесс, публикуешь результат

**Отсутствие клиентов — это сюжет**, а не проблема. Люди подписываются чтобы наблюдать путь к первому заработку.

## Монетизация

### Этап 1 — Услуги (старт)

**Что продаёшь:**
- Telegram-боты
- Лендинги
- Мини-аппы
- Автоматизация заявок

**Как конвертировать:**
В конце каждого поста про проект — короткий CTA: «Нужно такое же для твоего бизнеса — пиши в личку». Без агрессивных продаж.

**Ценник:**
5 000–15 000 ₽ за простой бот или лендинг. Ниже рынка на старте — чтобы взять первых клиентов и получить отзывы. Потом поднимать.

### Этап 2 — Партнёрки (через 1-2 месяца)

Только инструменты которые реально используешь:

| Инструмент | Зачем |
|------------|-------|
| TimeWeb | Хостинг, есть партнёрка |
| Beget | Хостинг, есть партнёрка |
| Supabase | БД, есть реферальная программа |
| VPN-сервисы | Высокий спрос, хорошие комиссии |
| Claude / Anthropic | API для разработчиков |

**Принцип:** пишешь гайд про инструмент → внизу реферальная ссылка. Не реклама, а «вот чем пользуюсь сам».
