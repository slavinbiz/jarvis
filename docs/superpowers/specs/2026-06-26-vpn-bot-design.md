## Архитектура (секция 1 — утверждена)

**Стек:** Python + aiogram 3.x + SQLite + s-ui API

**Файловая структура:**
```
vpn-bot/
├── bot.py          # точка входа, роутеры
├── db.py           # SQLite: таблицы users, subscriptions
├── sui.py          # обёртка над s-ui API
├── scheduler.py    # cron: проверка истёкших триалов и подписок
└── .env            # токен бота, данные s-ui, chat_id админа
```

**Состояния пользователя:**
```
new → trial (3 дня) → trial_expired → active → expired
```

Переходы делает scheduler каждый час. Выдача и отзыв — через s-ui API.

## Секция 2: Пользовательский флоу

### /start
Приветствие + инлайн-клавиатура:
- **[🚀 Получить VPN]** — основной CTA
- **[ℹ️ О сервисе]** — что такое, зачем, Hysteria2

### Кнопка [Получить VPN]
| Статус пользователя | Действие |
|---------------------|----------|
| `new` | Предложить триал 3 дня → создать в s-ui → отправить конфиг |
| `trial` | Показать статус + сколько дней осталось + конфиг |
| `trial_expired` | «Триал закончился» → предложить тарифы |
| `active` | Показать статус + дату окончания + конфиг |
| `expired` | «Подписка истекла» → предложить тарифы |

### Флоу оплаты
1. Пользователь выбирает тариф (кнопки: 1 мес / 3 мес / 12 мес)
2. Бот показывает сумму + реквизиты СБП (номер телефона)
3. Пользователь отправляет скрин/чек → бот пересылает админу с кнопками [✅ Подтвердить] [❌ Отклонить]
4. Админ нажимает [✅ Подтвердить] → бот активирует подписку + отправляет конфиг пользователю

### Конфиг
Отправляется как `vless://...` ссылка + QR-код (через qrcode библиотеку).
Инструкция: одна кнопка [📱 Как установить] → ссылка на README/Telegraph.

## Секция 4: Интеграция с s-ui

**Используем s-ui REST API** (не прямой SQLite — надёжнее).

### Аутентификация
```
POST /login  {"username": "...", "password": "..."}
→ Set-Cookie: session=...
```
Cookie сохраняем в памяти, переиспользуем. При 401 — переаутентифицируемся.

### Операции
| Действие | Запрос |
|----------|--------|
| Создать клиента | `POST /api/client` с `{email, enable: true, expiryTime: 0}` |
| Получить конфиг | `GET /api/inbounds` → найти клиента по email → собрать URI |
| Включить | `POST /api/client/updateClientIps/<email>` + `enable: true` |
| Отключить | Обновить клиента: `enable: false` |

**Идентификация клиентов в s-ui:** по полю `email` = `vpnbot_<user_id>`.

### Сборка конфига Hysteria2
URI формат: `hysteria2://<auth>@<host>:<port>/?insecure=1#Jarvis VPN`
Параметры берём из inbound-настроек панели.

## Секция 5: Scheduler

**Библиотека:** `APScheduler` (AsyncIOScheduler)

### Задачи (каждый час)
1. **Истечение триала:** найти `status=trial` где `trial_expires_at < now()` → disable в s-ui → статус `trial_expired` → уведомить пользователя
2. **Истечение подписки:** найти `status=active` где `expires_at < now()` → disable → статус `expired` → уведомить
3. **Напоминание:** найти `status=active` где `expires_at` через 1 день → отправить предупреждение (один раз, флаг `reminded=1`)

### Уведомления пользователю
- Триал истёк: «Ваш пробный период закончился. Подключите подписку — от 150 ₽/мес» + кнопка
- Подписка истекает завтра: «Завтра заканчивается VPN. Продлите чтобы не потерять доступ» + кнопка
- Подписка истекла: «Доступ отключён. Чтобы продолжить — выберите тариф» + кнопка
