# Telegram-канал: план запуска

> **Для выполнения:** прочитай спек `docs/superpowers/specs/2026-06-26-telegram-channel-strategy-design.md` перед началом.

**Цель:** Запустить Telegram-канал, опубликовать первые 10 постов, найти первых подписчиков.

**Архитектура:** Канал в нише «ИИ для мастеров и малого бизнеса», build-in-public стиль, монетизация через услуги → партнёрки.

**Стек:** Telegram, телефон с камерой (для скринов), Claude (для помощи с текстами при необходимости).

## Глобальные правила

- Тон: от первого лица, без пафоса, конкретные цифры и факты
- Никаких «экспертных» заявлений без доказательств — только личный опыт
- Каждый пост про проект заканчивается CTA: «Нужно такое же — пиши в личку»
- Длина поста: 5–15 предложений, без лишней воды

## Задача 1: Создать и оформить канал

**Файлы/артефакты:**
- Telegram-канал (публичный)
- Аватар (фото или иконка)
- Описание канала (до 255 символов)
- Закреплённый пост-знакомство

**Название канала — выбери одно из трёх:**

| Вариант | Почему |
|---------|--------|
| **«Сантехник с нейросетью»** | Максимально запоминается, вызывает любопытство (рекомендую) |
| «Мастер + ИИ» | Чёткое позиционирование для аудитории мастеров |
| «Код и трубы» | Краткое, игривое, но менее очевидное |

- [ ] **Шаг 1: Создать канал**
Telegram → Новый канал → Публичный → Выбрать username (например `@santex_ai` или `@slavin_ai`)

- [ ] **Шаг 2: Написать описание канала**

Вставь этот текст (или адаптируй):
```
Я сантехник. Строю продукты с ИИ без кода.
Канал про то, как обычный мастер зарабатывает на ИИ-инструментах.
Если ты думаешь «это не для меня» — я был там же.
```

- [ ] **Шаг 3: Поставить аватар**
Вариант А — твоё фото (лучше конвертирует, люди видят живого человека)
Вариант Б — иконка трубы + мозг/ИИ (если не хочешь светить лицом)

- [ ] **Шаг 4: Проверить**
Открой канал с другого устройства или через браузер — выглядит ли понятно кто ты и для кого канал?

### Пост 5 — Как это сделать: Telegram-уведомления за 15 минут

```
Самое полезное что я добавляю во все проекты — уведомления в Telegram.

Новая заявка на сайте → сообщение мне в Telegram.
Сигнал от крипто-бота → сообщение в Telegram.
Ошибка на сервере → сообщение в Telegram.

Делается за 15 минут:
1. Создать бота через @BotFather → получить токен
2. Узнать свой chat_id (написать боту /start, посмотреть через API)
3. Отправить POST запрос на:
   api.telegram.org/bot{TOKEN}/sendMessage

Всё. Больше не нужно заходить на сайт чтобы проверить заявки.

Нужна помощь с настройкой — пиши в личку.
```

## Задача 3: Найти первых подписчиков

Цель первого месяца — 100–200 подписчиков из целевой аудитории.
Не накрутка, живые люди которым это реально интересно.

**Неделя 1–2: личная сеть**

- [ ] Отправить ссылку на канал знакомым мастерам и предпринимателям (личное сообщение, не спам)

Текст для знакомых (личное сообщение, не в общий чат):
```
Привет! Завёл канал в Телеграме — рассказываю как я, сантехник,
без айтишного бэкграунда строю сайты, ботов и VPN с помощью ИИ.
Показываю всё как есть — код, косяки, деньги.

Если интересно — @santex_s_ai. Буду рад, если заглянешь.
```

- [ ] Написать в чат курса Дмитрия Ледовских — там твоя аудитория (вайбкодеры, начинающие)

Текст для чата курса (аудитория уже в теме, можно конкретнее):
```
Ребят, после курса довёл до продакшена несколько штук —
VPN-бот с рефералкой и автовыдачей конфига, лид-машину через
Telethon (сама ищет клиентов в Telegram), лендинг с формой на Supabase.

успехи, но и баги и как их чинил. Если интересно — @santex_s_ai
```

- [ ] Рассказать коллегам-сантехникам: «веду канал про ИИ»

Текст для коллег (устно или сообщением, попроще):
```
Я кроме труб теперь ещё каналом обзавёлся — показываю как обычному
мастеру без айтишного образования получается делать сайты и ботов
через нейросети. Если любопытно — @santex_s_ai
```

**Неделя 2–4: активность в чужих чатах**

Найти и вступить в эти типы Telegram-чатов (поиск делать самому в приложении Telegram — вступление и участие должно быть от твоего имени, не автоматизируется):
- [ ] Чаты по вайбкодингу и ИИ — искать: «вайбкодинг чат», «Claude Code чат», «нейросети для бизнеса»
