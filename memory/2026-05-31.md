## Сессия 3

Тема: обновление бота Jarvis на сервере, индикатор обработки

### Сделано
- Убрали вращающиеся фразы "Думаю… 🤔" из бота (токены тратились зря)
- Добавили статичный `⏳` пока бот обрабатывает запрос
- Настроили единую ветку `main` на GitHub (удалили `master`)
- Синхронизация памяти: VPS пушит каждые 5 мин, локалка тянет на старте сессии
- Обновили сервер (git reset --hard + systemctl restart)
- Сервер подвисал — перезапустили через Beget панель

### Решения
- `git reset --hard origin/main` надёжнее `git pull` при расхождении веток
- `su - agent -s /bin/bash -c "..."` — обходит dubious ownership при запуске от root
- Статичный `⏳` лучше тишины: пользователь видит что бот работает
