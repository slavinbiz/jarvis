### Архитектура бота
- `db.py` — SQLite (vpn_bot.db): таблицы users, payments
- `sui.py` — прямой доступ к s-ui SQLite `/usr/local/s-ui/db/s-ui.db` (REST API s-ui не авторизовался — endpoint `/app/api/login`)
- `scheduler.py` — APScheduler: истечение триала/подписки, напоминания за день
- `bot.py` — aiogram 3.x: флоу пользователя, оплата скриншотом, кнопки [✅/❌] у админа

### Ключевые детали
- s-ui API endpoint: `http://localhost:2095/app/api/login` (порт 2095, prefix /app/)
- s-ui авторизация через API не работала → используем прямой SQLite
- Hysteria2 inbound ID=1, порт 443, тег `hysteria2-443`
- После изменений в SQLite — `systemctl restart s-ui` (функция `_reload_sui()` в sui.py)
- Клиенты s-ui идентифицируются по полю `name` = `vpnbot_<user_id>`
