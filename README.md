# jarvis-architect

Шаблон архитектуры личного Агента в Claude Code. Часть курса **«Архитектор нейросотрудников»** Дмитрия Ледовских.

## Что это такое

Готовый набор файлов, из которых собирается ваш персональный Агент. Архитектура — три уровня памяти:

**Главные файлы Агента (всегда читаются)**

| Файл | Что внутри |
|---|---|
| `CLAUDE.md` | Правила работы Агента в этой папке. Claude Code читает первым. |
| `SOUL.md` | Личность Агента: тон, ценности, что любит / не любит. |
| `MEMORY.md` | Долгосрочная память: профиль, проекты, инфраструктура, стек, предпочтения. |
| `GOALS.md` | Цели на квартал и активные направления. |

**Папки рабочей памяти (создаются и наполняются по ходу)**

| Папка | Что внутри |
|---|---|
| `memory/` | Дневники работы по дням (`YYYY-MM-DD.md`). Агент ведёт сам. |
| `knowledge/` | Справочники и инструкции (брендбук, чек-листы, гайды). Вы кладёте сюда вручную или просите Агента собрать. |

**Telegram-бот (опционально)**

| Папка | Что внутри |
|---|---|
| `bot/` | Telegram-бот на Grammy + Claude Code CLI. Текст, голосовые, фото, документы, видео. Поддержка медиагрупп (несколько фото за раз). Работает 24/7. |

## Урок 4. Установка архитектуры (локально)

Откройте Claude Code в VS Code в папке вашего проекта и скажите:

> **Установи мне архитектуру Агента из github.com/Ntmib/jarvis-architect**

Claude сам:
1. Скачает шаблон (`npx degit Ntmib/jarvis-architect .`)
2. Задаст 10 коротких вопросов: имя, чем занимаетесь, цели, стиль общения
3. Заполнит файлы вашими ответами
4. Покажет превью — вы подтвердите или попросите поправить
5. Сделает первый коммит

Время: 10–12 минут.

## Урок 7. Перенос на сервер (VPS)

У вас уже есть заполненные файлы Агента на компьютере. Теперь переносим всё на сервер, чтобы Агент работал 24/7.

### Что нужно

- VPS на Beget (минимум 2 ГБ RAM)
- IP-адрес и root-пароль (получили при регистрации)
- VS Code с Claude Code на вашем компьютере

### Вариант А: Только файлы (без Telegram-бота)

Откройте Claude Code в VS Code **в папке вашего Агента** (там где лежат SOUL.md, MEMORY.md и т.д.) и вставьте этот текст, заменив IP и пароль на свои:

```
Помоги мне перенести моего агента на сервер.
IP моего сервера: 123.45.67.89
Мой пароль root: ваш_пароль
```

### Вариант Б: Полная установка с Telegram-ботом

Если хотите сразу запустить Telegram-бота на сервере:

```
Помоги мне перенести моего агента на сервер и поставить Telegram-бота.
IP моего сервера: 123.45.67.89
Мой пароль root: ваш_пароль
Мой токен бота от BotFather: 7123456789:AAHxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**Где взять токен бота:**
- Откройте Telegram, найдите @BotFather
- Отправьте команду /newbot, следуйте инструкциям
- Скопируйте токен (длинная строка с двоеточием)

Бот работает через вашу подписку Claude — API-ключ не нужен. Во время установки Claude попросит авторизоваться через браузер (один раз).

Замените `123.45.67.89` на IP вашего сервера и `ваш_пароль` на пароль от VPS.

Claude сам:
1. Подключится к серверу
2. Установит Node.js, Claude Code CLI
3. Создаст рабочие папки
4. Скопирует ваши файлы Агента на сервер
5. Установит и запустит Telegram-бота (если указали токены)
6. Скажет что делать дальше (настройка VS Code Tunnel)

### Настройте VS Code Tunnel

Claude даст инструкцию. Если коротко:
1. Зайдите в панель Beget → ваш VPS → **Консоль**
2. Введите: `code tunnel --accept-server-license-terms`
3. Откроется ссылка — перейдите по ней и введите код
4. Вернитесь в консоль, нажмите Ctrl+C
5. Введите: `code tunnel service install --accept-server-license-terms`

### Подключитесь из VS Code

1. Откройте VS Code на компьютере
2. Слева найдите **Remote Explorer** (иконка монитора)
3. В разделе **Tunnels** появится ваш сервер — нажмите на него
4. Готово — вы работаете на сервере как в обычной папке

## Если Claude не может подключиться к серверу

Если у вас VPN, он может блокировать подключение. Запасной путь:

1. Зайдите в панель Beget → VPS → **Консоль (VNC)**
2. Введите логин `root` и пароль
3. Введите две команды:

```
wget ntmib.github.io/jarvis-architect/s
bash s
```

Скрипт установит систему. После этого вернитесь в Claude Code и скажите:

```
Система на моём сервере уже установлена через VNC-консоль.
Помоги скопировать файлы агента на сервер и проверить что всё работает.
IP моего сервера: 123.45.67.89
Мой пароль root: ваш_пароль
```

│   └── INSTALL-SERVER.md  ← инструкция для Claude: установка на VPS

├── docs/

        ├── README.md       ← гайд по скиллам (урок 5)
        ├── server-setup/
        │   └── SKILL.md    ← скилл установки на VPS

            └── SKILL.md    ← дизайн интерфейсов
